# 01. 左值右值

## <font color=cyan>01. 定义</font>

|         |        |                                          |                                                              |
| ------- | :----- | :--------------------------------------- | :----------------------------------------------------------- |
| lvalue  | 左值   | 有可访问的内存地址                       | 变量名，const变量，数组元素，返回左值引用的函数调用，类成员  |
| prvalue | 纯右值 | 没有可访问的内存地址                     | 字面值常量，返回非引用类型的函数调用，在表达式求值期间创建但是只能被编译器访问的临时对象 |
| xvalue  | 将亡值 | 有地址但不会再被访问，可以初始化右值引用 | 返回右值引用的函数调用，数组下标                             |
